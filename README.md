# Java Fundamentos

Repositório com exercícios práticos desenvolvidos durante meus estudos de Java.

## Objetivo

Praticar e consolidar os principais conceitos da linguagem Java, incluindo:

- Lógica de programação
- Estruturas condicionais
- Estruturas de repetição
- Programação Orientada a Objetos (POO)
- Encapsulamento
- Arrays e Listas

---

## Estrutura do Projeto

### 01 - Lógica Básica
Exercícios de:
- Entrada e saída de dados
- Variáveis
- If/Else
- Switch
- While
- For
- Cálculos matemáticos

### 02 - POO e Encapsulamento
Exercícios com:
- Classes
- Objetos
- Métodos
- Encapsulamento
- Organização em pacotes

### 03 - Arrays e Listas
Exercícios envolvendo:
- Vetores
- Manipulação de arrays
- Estruturas de repetição aplicadas
- Classes com arrays
